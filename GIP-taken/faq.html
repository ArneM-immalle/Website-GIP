Résolvez vos doutes en consultant les questions les plus courantes.

Comment est-ce que je peux créer un class notebook ?

Si vous créez un team en Teams, un class notebook est créé automatiquement. Pour créer un team, appuyez d'abord deux fois sur le bouton "Teams". Ensuite, appuyez sur le bouton "Devenez un membre ou créez un Team". Après choisissez un Team pour les groupes de leçons et suivez les instructions. Enfin, appuyez sur "Créez un Class Notebook" et suivez les instructions.

Où puis-je noter le matériel de cours ?

Notez tout le matériel de cours (que les étudiants sont seulement autorisés à lire) dans la section group "_Content Library".

Mes étudiants peuvent collaborer à un projet où?

Les élèves peuvent collaborer à un projet dans la section group "_Collaboration space". Ici, ils ont le droit de lire et d'écrire.

Puis-je donner des tâches individuelles aux étudiants ?

Oui, lors de la création d'un Class Notebook, une section group est créé pour chaque élève. Seuls vous et l'étudiant spécifique avez accès à ce section group. Si vous souhaitez qu'une page soit remplie individuellement par vos élèves, vous cliquez sur distribuer sous l'onglet Class Notebook. De cette façon, cette page est créée dans la section individuelle de chaque élève.
