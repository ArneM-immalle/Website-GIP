Comment créer un class notebook ?

Si vous créez un team en Teams, un class notebook est créé automatiquement. Pour ce faire, appuyez d'abord deux fois sur le bouton "Teams". Ensuite, appuyez sur le bouton "Devenir un membre ou créer un Team". Après choisissez un Team pour les groupes de leçons et suivez les instructions. Enfin, appuyez sur "Créez un Class Notebook" et suivez les instructions.

Où puis-je noter le matériel de cours ?

Notez tout le matériel de cours (que les étudiants sont seulement autorisés à lire) dans le section group "_Content Library".

Mes étudiants peuvent collaborer à un projet où?

Les élèves peuvent collaborer à un projet dans le section group "_Collaboration space". Ici, ils ont le droit de lire et d'écrire.

Puis-je donner des tâches individuelles aux étudiants ?

Oui, lors de la création d'un Class Notebook, un section group est créé pour chaque élève. Seuls vous et l'étudiant en question avez accès à ce section group.

Comment assigner des tâches aux étudiants ?

Si vous souhaitez qu'une page soit remplie individuellement par vos élèves, cliquez sur distribuer sous l'onglet Class Notebook. De cette façon, cette page est créée dans la section individuelle de chaque élève.
